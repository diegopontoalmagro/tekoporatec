Configurar sincronização
Seus dados serão salvos no seu GitHub e sincronizados em qualquer dispositivo automaticamente.

💡 Ainda não tem repositório? Acesse github.com/new, crie um com o nome tekoporatec (público ou privado) e volte aqui.

Chave de acesso
Crie um token no GitHub para autorizar o salvamento automático. É feito uma única vez.
  1. Acesse github.com/settings/tokens/new
  2. Em Note: escreva TekoporãTec
  3. Em Expiration: escolha No expiration
  4. Marque a caixa repo (primeira da lista)
  5. Clique em Generate token no final da página
  6. Copie o token (começa com ghp_) e cole abaixo
🎉
