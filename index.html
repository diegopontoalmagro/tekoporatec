Passo 1 de 3 — Repositório no GitHub
Onde salvar seus dados?
Seus dados serão salvos como um arquivo no seu GitHub.
Qualquer dispositivo que abrir o site carrega os mesmos dados automaticamente.
💡 Se ainda não criou o repositório, acesse github.com/new, crie um repositório com o nome que quiser (ex: tekoporatec) e volte aqui.
Chave de acesso (Token)
O token permite que o app salve dados no seu repositório. Você cria uma vez e nunca mais precisa mexer.
  1. Acesse github.com/settings/tokens/new
  2. Em Note escreva: TekoporãTec
  3. Em Expiration escolha: No expiration
  4. Marque a caixa repo (primeira da lista)
  5. Clique em Generate token no final da página
  6. Copie o token gerado (começa com ghp_) e cole abaixo
