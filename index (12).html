Sem repositório? Acesse github.com/new, crie um chamado tekoporatec e volte aqui.

Chave de acesso
Crie um token no GitHub uma única vez para autorizar o salvamento automático.
  1. Acesse github.com/settings/tokens/new
  2. Em Note: TekoporãTec
  3. Em Expiration: No expiration
  4. Marque a caixa repo (primeira da lista)
  5. Clique em Generate token no final
  6. Copie o token (começa com ghp_) e cole abaixo
